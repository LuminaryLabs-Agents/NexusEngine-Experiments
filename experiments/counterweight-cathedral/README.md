# Counterweight Cathedral

SAT-005 composes pinned NexusEngine Core with Physics Body Lite and Weighted Trigger. Move three stones along a rail and settle exact weights `2 / 3 / 1` into the three suspended bowls.

Serve the repository root and open `/experiments/counterweight-cathedral/`.

Controls: `Q/E` select stone, `1` shift left, `2` shift right, `3` settle, `R` restart.
