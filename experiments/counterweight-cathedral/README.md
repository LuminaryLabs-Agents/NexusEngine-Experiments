# Counterweight Cathedral

SAT-005 composes pinned NexusEngine Core with Physics Body Lite and Weighted Trigger. Move three stones along a rail and settle exact weights `2 / 3 / 1` into the three suspended bowls.

The route now consumes `application-domain-kit.js`, a frozen local full-game descriptor that owns authored fantasy, scene inventory, tuning, policies, selected domains, and proof budgets. It contains no reusable physics/trigger implementation and no browser or renderer adapter.

The selected stone's matching bowl is identified in the hero prompt and highlighted in both the sculpture and bowl strip, so exact placement is readable without opening diagnostics.

Serve the repository root and open `/experiments/counterweight-cathedral/`.

Controls: `Q/E` select stone, `1` shift left, `2` shift right, `3` settle, `R` restart.

Forward stage: `validated playable -> application-domain kit`. The next separate stage maps its atomic responsibilities; no promotion or purge is implied by this packaging step.
