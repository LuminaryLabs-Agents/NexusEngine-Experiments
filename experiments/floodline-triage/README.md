# Floodline Triage

SAT-001 is a bounded playable route-domain stress experiment for the consolidated Nexus Engine atlas.

Run the repository from its root and open `/experiments/floodline-triage/`.

The browser host composes the pinned NexusEngine realtime/DSK substrate with the pinned Navigation Knowledge, Rescue Triage, and Structural Support Network ProtoKits. Scenario content, controls, ending rules, and Canvas/DOM presentation remain local.
