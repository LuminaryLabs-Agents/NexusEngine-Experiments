# Luminous Nursery

SAT-003 composes pinned NexusEngine Core with Habitat Suitability and Ecosystem Population Cycle. Sample and repair three reef cradles, seed only eligible cohorts, then pulse two deterministic seasons.

Serve the repository root and open `/experiments/luminous-nursery/`.

Controls: `Q/E` select, `1` sample, `2` tune, `3` seed, `4` pulse season, `R` restart.
