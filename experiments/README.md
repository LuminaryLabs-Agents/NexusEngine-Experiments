# Experiments

Playable NexusRealtime validation apps for the arcade.

Use this folder for games that specifically prove runtime composition, stable kits, ProtoKits, sequence flow, renderer boundaries, input adapters, or promotion candidates.

Each experiment should have:

```txt
index.html
README.md
src/
```

The root `index.html` is the arcade lobby and should include a card for each playable experiment.
