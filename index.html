Why this page exists

The root should be the arcade lobby. Individual games stay isolated in their own folders so assets, level data, renderer code, and experiment docs remain close to the game.

How to add the next game

Create a new folder under experiments/ or games/, make it playable from index.html, then add one card here.

Deployed from main with GitHub Pages. Static arcade, no build step required.
