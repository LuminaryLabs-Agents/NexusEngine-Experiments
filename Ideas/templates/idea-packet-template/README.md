# Idea Packet Template

Copy this folder when creating a new game idea packet.

Rename the copied folder using lowercase hyphenated words.

Example:

```txt
Ideas/packets/neon-orbit-racer/
```

## Packet files

```txt
README.md
concept.md
prototype-plan.md
scoring-rubric.md
playtest-notes.md
```

## Packet status

Set one status in the packet README:

```txt
raw
refined
ready-for-prototype
playtest-candidate
production-candidate
merged
retired
superseded
shipped
```
