# Nexus Arcade Game Ideas

This repository is the planning home for Nexus Arcade game concepts.

We will load game ideas here as small, organized documents instead of leaving them scattered across chats, notes, and experiments.

## Purpose

- Capture arcade-ready game ideas quickly.
- Keep each idea easy to review, remix, and prioritize.
- Separate rough concepts from production-ready game plans.
- Build a shared backlog for Nexus Arcade content.

## Planned folder structure

```txt
NexusArcade-GameIdeas/
├─ README.md
├─ README.pdf
├─ ideas/
│  ├─ raw/
│  │  └─ quick one-page idea drops
│  ├─ refined/
│  │  └─ clearer playable concepts with mechanics and loops
│  └─ ready-for-prototype/
│     └─ ideas scoped tightly enough to build
├─ genres/
│  ├─ action/
│  ├─ puzzle/
│  ├─ racing/
│  ├─ co-op/
│  ├─ rhythm/
│  └─ experimental/
├─ templates/
│  └─ game-idea-template.md
└─ archive/
   └─ retired, merged, or superseded concepts
```

## Idea document shape

Each game idea should eventually include:

- Title
- One-line pitch
- Arcade cabinet fit
- Core loop
- Controls
- Player goal
- Failure state
- Visual identity
- Audio identity
- Prototype scope
- Notes / open questions

## Current status

This repo is being prepared as the organized idea bank for Nexus Arcade.

Game ideas will be added incrementally into the folder structure above.
